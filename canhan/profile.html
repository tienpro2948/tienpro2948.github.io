📘 Quá trình học tập

Giai đoạn 1: Học lập trình C cơ bản

Bắt đầu với ngôn ngữ C, tôi nắm vững các khái niệm cơ bản như biến, kiểu dữ liệu, vòng lặp, điều kiện và con trỏ. Việc học C giúp tôi hiểu sâu về cách máy tính vận hành và lập trình hệ thống.

Giai đoạn 2: Phát triển kỹ năng Python

Hiện tại tôi tập trung phát triển kỹ năng Python, đặc biệt trong lĩnh vực tự động hóa và xử lý dữ liệu. Python giúp tôi làm việc nhanh chóng và hiệu quả hơn trong các dự án thực tế.

Giai đoạn 3: Học web cơ bản với HTML & CSS

Để hướng đến phát triển web, tôi đang học và thực hành HTML, CSS để xây dựng các giao diện web chuẩn, thân thiện với người dùng và chuẩn responsive.

Mục tiêu của tôi là kết hợp các kỹ năng này để phát triển các sản phẩm công nghệ có tính ứng dụng cao.

💡 Kỹ năng & Công nghệ

  • 🔥 Python: Thành thạo từ căn bản đến automation thực tế.
  • 💻 HTML + CSS: Thiết kế giao diện web responsive, hiện đại, tối ưu trải nghiệm người dùng.
  • 🧠 C cơ bản: Nền tảng lập trình hệ thống, tư duy cấu trúc dữ liệu và thuật toán.
  • 🚀 GitHub, VSCode: Công cụ phát triển và quản lý mã nguồn chuyên nghiệp.
  • 🎯 Tư duy logic & tự học: Khả năng phân tích, tìm hiểu và áp dụng kiến thức mới nhanh chóng.
