📘 Quá trình học tập

Tôi là sinh viên ngành Công nghệ Thông tin, bắt đầu với ngôn ngữ lập trình C cơ bản. Hiện tại, tôi đang phát triển chuyên sâu Python để phục vụ cho automation và các dự án thực tế. Ngoài ra, tôi cũng học thêm HTML và CSS để xây dựng các sản phẩm web chuyên nghiệp, đáp ứng tiêu chuẩn responsive hiện đại.

Mục tiêu của tôi là kết hợp kiến thức lập trình nền tảng với kỹ năng thiết kế giao diện, từ đó tạo ra những ứng dụng web thân thiện, dễ sử dụng và hiệu quả.

  • 🧠 C cơ bản – nền tảng lập trình hệ thống, giúp hiểu sâu về cấu trúc dữ liệu và thuật toán.
